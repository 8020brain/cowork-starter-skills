# CoWork Starter Skills

8 free skills for [Claude CoWork](https://cowork.claude.com) that solve real business problems.

## Skills

| Skill | What it does |
|-------|-------------|
| **search-term-analyzer** | Analyze Google Ads search terms. Find wasted spend, generate negative keyword lists, surface promotion candidates. |
| **csv-analyzer** | Drop in any CSV and get an interactive HTML report with charts, stats, and insights. Auto-detects data types. |
| **post-meeting** | Turn meeting transcripts into action items, decisions, and a follow-up email draft. |
| **inboxy** | Triage a batch of emails or messages by priority. Get a dashboard with draft replies. |
| **followups** | Scan your contacts for overdue relationships. Get personalised check-in drafts and an outreach calendar. |
| **draft-post** | Write polished LinkedIn or X/Twitter posts from rough ideas with platform-specific formatting. |
| **post-evaluator** | Score any post across 5 lenses (Clarity, Voice, Hook, Substance, Platform Fit) with rewrite suggestions. |
| **competitor-scraper** | Fetch competitor Google Ads and generate a competitive intelligence report. |

## Install

### Option 1: Add the marketplace (recommended)

In CoWork, go to **Customize > Skills > Browse plugins > Add a marketplace** and enter:

```
https://github.com/8020brain/cowork-starter-skills
```

Then install individual skills from the marketplace browser.

### Option 2: Download individual skills

Each skill folder is also available as a zip on the [Releases](https://github.com/8020brain/cowork-starter-skills/releases) page. Download and upload directly to CoWork.
